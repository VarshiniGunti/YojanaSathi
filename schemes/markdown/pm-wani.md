# PM-WANI - Public WiFi Scheme

## Category
WiFi, internet, entrepreneurship, digital India, self-employed

## Ministry
Department of Telecommunications

## Description
Promotes setting up of public WiFi hotspots by entrepreneurs. Provides framework for public data office aggregators to provide internet services.

## Eligibility Criteria

### Profession
Self-Employed, Unemployed, Student

### Category/Caste
General, OBC, SC, ST, EWS

### Age Requirement
18 to 100 years

### Income Limit
Above ₹50,000 / month

### Additional Conditions
- Must be willing to set up public WiFi hotspot
- No license fee required

## Benefits
- No license fee for providing WiFi service
- Low cost of setting up hotspot
- Revenue sharing model
- Support from PDO aggregators
- Growing demand for public WiFi
- Digital India initiative support

## Required Documents
1. Aadhaar Card
2. PAN Card
3. Bank Account Passbook
4. Address Proof

## Application Process
1. Register with any PDO Aggregator
2. Purchase WiFi equipment
3. Set up public WiFi hotspot
4. Start providing internet services
5. Earn revenue from users

## State Applicability
